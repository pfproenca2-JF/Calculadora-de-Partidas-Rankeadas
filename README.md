# Calculadora de Partidas Rankeadas 🎮

Desafio de projeto do curso de lógica de programação da DIO.

## 🛠️ Tecnologias Utilizadas
- JavaScript
- Node.js (para execução)

## 📝 Objetivo
Criar uma lógica que calcule o saldo de vitórias e classifique o nível do jogador com base em faixas de pontuação predefinidas.

## 🧠 Conceitos Aplicados
- Funções com parâmetros e retorno
- Estruturas condicionais (if/else if)
- Laços de repetição (for)
- Template Strings para saída formatada
